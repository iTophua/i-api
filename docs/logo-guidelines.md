## 1. 品牌概述

### 品牌名称
**iApi**

### 品牌定位
轻量、快速、中文的 API 测试工具

### 核心价值
- **轻量**：体积小（< 20MB），无臃余功能
- **快速**：启动快（< 2秒），响应迅速
- **专业**：专注 API 测试，功能精准
- **本地**：纯本地运行，数据隐私可控
- **中文**：原生中文界面，无翻译感

### 目标用户
国内开发者、API 开发和测试人员

## 2. Logo 设计理念

## 3. Logo 版本

### 3.1 主 Logo（横版）

适用于网站头部、文档标题、宣传材料等。

**文件：**
- `logo-main.svg` - 主版本
- `logo-horizontal-light.svg` - 浅色背景版
- `logo-horizontal-dark.svg` - 深色背景版

**规格：**
- 宽高比：约 3.3:1
- 推荐最小宽度：120px

### 3.2 图标 Logo（方形）

适用于应用图标、网站 favicon、社交媒体头像等。

**文件：**
- `icon.svg` - 标准图标（60×60）
- `app-icon.svg` - 应用图标（512×512）
- `favicon.svg` - 网站图标（32×32）
- `icon-mono.svg` - 单色版本

**规格：**
- 宽高比：1:1
- 圆角半径：约 23%（14px/60px）

### 3.3 Logo 变体对照表

| 版本 | 用途 | 格式 | 推荐尺寸 |
|------|------|------|----------|
| 主 Logo | 网站、文档、宣传 | SVG/PNG | 200-400px 宽 |
| 图标 | 应用图标、头像 | SVG/PNG | 60-512px |
| Favicon | 浏览器标签页 | SVG/PNG/ICO | 16-32px |
| 单色版 | 单色印刷、水印 | SVG/PNG | 按需 |

## 4. 色彩规范

### 4.1 主色（紫色渐变）

| 色彩名称 | HEX | RGB | 用途 |
|----------|-----|-----|------|
| 紫色（起始） | `#7C3AED` | 124, 58, 237 | 渐变起始色 |
| 紫色（结束） | `#A855F7` | 168, 85, 247 | 渐变结束色 |

**CSS 渐变：**
```css
background: linear-gradient(135deg, #7C3AED 0%, #A855F7 100%);
```

### 4.2 辅助色（金色渐变）

| 色彩名称 | HEX | RGB | 用途 |
|----------|-----|-----|------|
| 金色（起始） | `#FCD34D` | 252, 211, 77 | 闪电渐变起始 |
| 金色（结束） | `#F59E0B` | 245, 158, 11 | 闪电渐变结束 |

**CSS 渐变：**
```css
background: linear-gradient(135deg, #FCD34D 0%, #F59E0B 100%);
```

### 4.3 文字色

| 场景 | HEX | RGB |
|------|-----|-----|
| 浅色背景文字 | `#1F2937` | 31, 41, 55 |
| 深色背景文字 | `#FFFFFF` | 255, 255, 255 |

### 4.4 深色主题变体

用于深色背景时，logo 颜色适当提亮：

| 色彩名称 | HEX | RGB |
|----------|-----|-----|
| 紫色（浅） | `#A78BFA` | 167, 139, 250 |
| 紫色（更浅） | `#C4B5FD` | 196, 181, 253 |

## 5. 最小尺寸

为确保 logo 清晰可辨，请遵守以下最小尺寸限制：

### 5.1 横版 Logo

| 应用场景 | 最小宽度 | 最小高度 |
|----------|----------|----------|
| 网站展示 | 120px | 36px |
| 打印材料 | 30mm | 9mm |
| 移动端 | 100px | 30px |

### 5.2 图标 Logo

| 应用场景 | 最小尺寸 |
|----------|----------|
| 应用图标 | 16×16px |
| 网站图标 | 16×16px |
| 社交媒体 | 48×48px |
| 打印材料 | 10×10mm |

### 5.3 特殊情况

当尺寸接近最小限制时：
- 优先使用图标版本而非横版
- 确保闪电元素清晰可见
- 避免在复杂背景上使用

## 6. 安全区域

### 6.1 定义

安全区域是指 logo 周围必须保留的最小空白区域，确保 logo 不被其他元素干扰。

### 6.2 横版 Logo 安全区域

```
┌──────────────────────────────────────┐
│              安全区域                 │
│    ┌─────────────────────────┐      │
│    │                         │      │
│ 安 │      iApi Logo          │ 安   │
│ 全 │                         │ 全   │
│ 区 │                         │ 区   │
│    └─────────────────────────┘      │
│              安全区域                 │
└──────────────────────────────────────┘
```

**安全区域大小：** Logo 高度的 25%

### 6.3 图标 Logo 安全区域

```
┌────────────────┐
│    安全区域    │
│  ┌──────────┐  │
│  │          │  │
│  │  Icon    │  │
│  │          │  │
│  └──────────┘  │
│    安全区域    │
└────────────────┘
```

**安全区域大小：** 图标尺寸的 12.5%

### 6.4 安全区域计算示例

| Logo 尺寸 | 安全区域 |
|-----------|----------|
| 200×60px | 15px |
| 100×30px | 7.5px |
| 60×60px | 7.5px |
| 32×32px | 4px |

## 7. 禁用形式

### 7.1 ❌ 禁止事项

| 禁止行为 | 说明 | 错误示例 |
|----------|------|----------|
| 拉伸变形 | 不得非等比缩放 | ⬜️→🟫 |
| 旋转倾斜 | 不得旋转或倾斜 logo | 🔃 |
| 改变颜色 | 不得使用非规范颜色 | 🎨 |
| 添加效果 | 不得添加阴影、描边等 | 🌟 |
| 背景冲突 | 不得在复杂背景上使用 | 🌈 |
| 元素分离 | 不得分离 logo 组成部分 | ✂️ |
| 模糊处理 | 不得使用模糊效果 | 🌫️ |
| 透明度调整 | 不得随意调整透明度 | 👻 |

### 7.2 ❌ 错误示例

```
❌ 拉伸：  [iApi] → [iApi    ]
❌ 压缩：  [iApi] → [iApi]
❌ 旋转：  [iApi] ↻
❌ 变色：  [iApi] (绿色/红色等非规范色)
❌ 加边框： ⬜[iApi]⬜
❌ 加阴影： [iApi] with drop shadow
```

### 7.3 ✅ 正确做法

```
✅ 等比缩放：保持宽高比
✅ 使用规范色：紫色渐变 + 金色闪电
✅ 保持安全区域：周围留白充足
✅ 选择合适背景：纯色或简单渐变背景
✅ 使用完整 logo：不拆分元素
```

## 8. 应用场景

### 8.1 应用图标

**平台：** macOS、Windows

**要求：**
- 使用 `app-icon.svg` 或对应 PNG 文件
- macOS：512×512px，圆角自动处理
- Windows：256×256px 或更小尺寸

**文件：**
- `src-tauri/icons/icon.png` - 1024×1024
- `src-tauri/icons/icon.icns` - macOS 图标集
- `src-tauri/icons/icon.ico` - Windows 图标

### 8.2 网站应用

#### 网站头部
- 使用横版 logo
- 高度建议：36-48px
- 链接至首页

#### Favicon
- 使用 `favicon.svg` 或 `favicon-32x32.png`
- 支持深色模式适配

```html
<link rel="icon" type="image/svg+xml" href="/favicon.svg">
<link rel="icon" type="image/png" sizes="32x32" href="/favicon-32x32.png">
<link rel="apple-touch-icon" href="/apple-touch-icon.png">
```

### 8.3 社交媒体

| 平台 | 推荐尺寸 | 使用版本 |
|------|----------|----------|
| GitHub | 256×256px | 图标版 |
| Twitter | 400×400px | 图标版 |
| 微信公众号 | 300×300px | 图标版 |
| 掘金 | 200×200px | 图标版 |

### 8.4 宣传材料

#### 横幅/Banner
- 使用横版 logo
- 背景简洁，突出 logo
- 保留安全区域

#### 海报
- Logo 尺寸根据海报大小调整
- 确保最小尺寸要求
- 高分辨率输出（300dpi）

#### 名片
- Logo 宽度：20-25mm
- 使用横版或图标版
- 确保印刷质量

### 8.5 文档与演示

#### 文档头部
- 使用横版 logo
- 高度：30-40px
- 左对齐或居中

#### PPT 演示
- 封面：Logo 居中或右上角
- 内页：Logo 可缩小放于角落
- 保持品牌一致性

### 8.6 主题适配

#### 浅色主题
```
背景：#FFFFFF 或浅灰色
Logo：logo-horizontal-light.svg
文字：#1F2937
```

#### 深色主题
```
背景：#1F2937 或深色
Logo：logo-horizontal-dark.svg
文字：#FFFFFF
```
